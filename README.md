# Aurora-LINK
## Présentation

Aurora-LINK est une application qui permet de configurer et monitorer les appareils Aurora (https://github.com/LOGDrakon/Aurora). L'application permet donc de configurer les entrées de l'appareil, des scènes. Elle permet aussi de monitorer les entrées physiques et d'analyser le comportement de l'appareil.
Aurora-LINK est basé sur le protocole LINK (https://github.com/LOGDrakon/LINK-Client) qui permet de faciliter la gestion de la communication avec un appareil.

## Requiert
Windows 10/11. Pour l'instant il n'existe pas de version Linux ou MacOS
